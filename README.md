# NetWorthApp

Простой SwiftUI-приложение для iOS, которое помогает отслеживать чистый капитал пользователей Казахстана.

## Возможности
- Ведение активов (наличные, депозиты, инвестиции KASE/AIX/международные, недвижимость, бизнес)
- Учёт обязательств (Kaspi кредиты, ипотека, прочие долги)
- Подсчёт чистого капитала в KZT и USD
- История чистого капитала и график динамики
- Лестница богатства Казахстана с подсветкой текущей ступени
- AI-подсказки по росту капитала
- Прогноз выхода из долгов
- Переключение языка интерфейса (русский, казахский)

## Требования
- Xcode 14.3+
- iOS 16+ (для использования Swift Charts)

## Запуск
1. Откройте папку `NetWorthApp` в Xcode как SwiftUI-проект (создайте новый проект и перенесите файлы или добавьте их в существующий).
2. Запустите на симуляторе или устройстве.

## Локализация
Строки интерфейса расположены в `Resources/ru.lproj` и `Resources/kk.lproj`.
